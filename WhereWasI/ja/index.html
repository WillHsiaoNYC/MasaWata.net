訪れた場所をすべて記録

WhereWasI は訪れた場所を自動で追跡・整理します。お気に入りの場所を振り返り、日常のパターンを発見し、プライベートな旅の記録を残しましょう——すべて手間いらずで。

パワフルな機能

位置履歴を記憶し探索するために必要なすべて

あなたの人生をタイムラインで

美しく整理された訪問履歴をスクロール。日、週、月ごとに訪問を確認——あなたの人生の物語が自動的に展開されます。

パターンを発見

日常生活についての洞察を得ましょう。最も訪れた場所、各場所で過ごした時間、日課がどのように変化しているかを確認できます。

世界を一目で

すべての訪問をインタラクティブな地図で確認。ズームアウトして人生があなたを連れて行ったすべての場所を見たり、ズームインして一つの地域を探索したり。

データはあなたのデバイスに

すべての位置データは iPhone と個人の iCloud に安全に保存されます。外部サーバーに送信されることはありません。あなたの旅はあなただけのものです。

美しいデザインとパワフルな機能の融合

今日から記録を始めよう

無料でダウンロード。インストールした瞬間からロケーション日記が始まります。アカウント登録不要——あなたと iPhone、そして訪れる場所だけ。

WhereWasI について知っておくべきすべてのこと

WhereWasI は iOS の位置情報サービスを使用して、場所への到着と出発を検出します。バックグラウンドで効率的に動作し、バッテリーを消耗しません。

はい、完全に安全です。すべてのデータはデバイスにローカル保存され、個人の iCloud アカウントにのみ同期されます。外部サーバーで位置データを閲覧したり保存したりすることはありません。

もちろんです！完全なコントロールが可能です。訪問時間の編集、重複した場所の統合、メモの追加、保存したくない訪問の削除ができます。

いいえ。WhereWasI は Apple の省電力型重要位置変更 API を使用しており、バッテリーへの影響を最小限に抑えながら訪問を正確に記録するよう設計されています。

プライバシーを大切に

WhereWasI はすべての位置データをデバイスと個人の iCloud アカウントに保存します。あなたの旅の記録が私たちや第三者と共有されることはありません。
